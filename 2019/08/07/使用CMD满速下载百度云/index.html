使用CMD满速下载百度云

发表于 2019-08-07 更新于 2019-08-09 阅读次数：

工具准备

  1. windows 10电脑
  2. 下载工具：baidupcs-go
  3. win10打开CDM快捷键

    win+R后输入CMD回车

环境配置

  1. 桌面右击我的电脑→属性→高级系统设置→环境变量
  2. 在系统变量中选择Path变量→编辑→新建→输入你的BaiduPCS-Go的存放目录，注意是存放目录，例如我的是 D:\BaiduPCS-Go-v3.5.6-windows-x64，最后依次确认

    这样才可以使用CMD命令快速打开baidupcs-go

使用教程

登录

  1. 配置好环境变量后，打开cmd

    输入命令：baidupcs-go回车

  2. 登录你的百度网盘账号

    输入命令 login回车

    依次输入账户密码，注意CMD中输入密码是不会显示出来的（哪怕是*号也不会显示）。

  3. 复制路径在文件管理器中粘贴打开
  4. 选择验证方式

    文件下载

  5. 列出当前位置所有文件

    输入命令 ls 列出当前位置所有文件

  6. 选择目录（选择文件夹）

    cd 目录（只要输入前几个字按tab键就可以自动补全目录）

  7. 下载文件

    d 文件名

    总结

    至此，基本使用教程已经讲完了，也可以登录后输入命令‘help’查看命令集,如有其它问题
